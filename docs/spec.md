# Specs for this application

I have to be able to filter students based on their performance, specially if the belong to different groups
Today (April 15, 206) the Bachelors in Computer Science are separated in two groups, "G1" and "G2". I have the students data in csv files:

1. one with all the the students grades based on their performance
2. one separated by groups

The goal is to be able to compare grades among groups for any possible inconsitencies between groups, teachers, ways of evaluation, etc
If something like this is found, I need to contact the teacher and find a way to compensate the situation

## Tech stach

- Node JS rest API with Express for the server
- React for the front-end (tailwind css)
- Typescript for the types (backend and front-end too)

## Plan

- Create the API as an MVP, simple and easy to use, minimalistic. It could get more complex as the project grows
- Create the front-end as an MVP, simple and easy to use, minimalistic. It could get more complex as the project grows
