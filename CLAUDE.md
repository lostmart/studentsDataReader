# Students Filtering System

## Project Overview

A full-stack tool for filtering and comparing student grades across groups to detect inconsistencies between groups, teachers, or evaluation methods. When inconsistencies are found, the relevant teacher is contacted to compensate the situation.

## Data Sources

Two CSV files:
1. Student grades based on performance (all students)
2. Students separated by groups (currently "G1" and "G2" for Bachelors in Computer Science)

## Tech Stack

- **Backend:** Node.js REST API with Express, TypeScript
- **Frontend:** React with Tailwind CSS, TypeScript

## Architecture Principles

- MVP-first: keep it simple, minimal, and easy to use
- Avoid over-engineering — complexity grows only as the project requires it
